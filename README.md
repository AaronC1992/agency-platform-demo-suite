# Agency Platform Demo Suite

[![Open in StackBlitz](https://developer.stackblitz.com/img/open_in_stackblitz.svg)](https://stackblitz.com/github/AaronC1992/agency-platform-demo-suite)

> Click the button above to launch the app instantly in your browser — no installation required.

---

A white-label SaaS demo suite built for local marketing agencies. Features three fully interactive product demos you can walk clients through in meetings.

## What's Included

| Demo | Description |
|------|-------------|
| **AgencyOS** | Client portal with CRM, campaigns, reviews, referrals, and reporting |
| **LocalLead** | HVAC/service business CRM with leads, pipeline, estimates, and calendar |
| **LocalBoost** | Loyalty & rewards platform with QR tracking, redemptions, and referrals |

All demos use realistic Joplin/Southwest Missouri mock data. No real data is ever stored.

## Run Locally

```bash
npm install
npm run dev
```

Then open [http://localhost:5173](http://localhost:5173).

## Tech Stack

- React 18 + Vite 8
- Pure CSS (no UI library)
- State-based routing (no React Router)
- Mock data only — no backend required
